Doc Manager
===========

# Features

- 문서를 git에서 관리
- GIT에서 블로그로 자동 배포
- 대상 서비스가 API를 제공하는 경우는 API, 아니면 html붙여넣기

문서 아이디를 매핑해서 관리
- tistory
- wordpress
- ghpages
  기타 API지원 안되는 경우엔 수동으로
